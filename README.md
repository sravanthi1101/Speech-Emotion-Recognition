# Speech-Emotion-Recognition

## Speech based Emotion Recogntion built using Deep Neural Networks of RNN and MLP while exploring various prperties of Librosa

### Dataset
Toronto emotional speech set (TESS) from Kaggle

### Tech Stack
- Tensorflow
- Keras
- Pandas
- Numpy
- Scikit Learn
- Librosa
- Wavio
- SoundDevice

### Model Accuracy
RNN(LSTM)- 97.02%
MLP-98.5%
